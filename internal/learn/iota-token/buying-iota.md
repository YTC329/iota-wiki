# 購買 IOTA

IOTA 在大多數主要的加密貨幣交易所上市。

- [**Binance.US**](https://www.binance.us/en/home)
- [**Bittrex**](https://global.bittrex.com/)
- [**Gate.io**](https://www.gate.io/)
